# BoardArt

A board game tracking app built for artisans — designed to help track games, collections, and related data.

## About

This is a training project built with [Claude Code](https://claude.ai/claude-code), Anthropic's AI coding assistant. The goal is to explore AI-assisted development while creating a practical tool for board game artisans.

## What it does

- Track board games and collections
- Manage game-related data for artisans and creators
